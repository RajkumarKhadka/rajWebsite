# Raj Kumar Khadka — Personal Portfolio

Personal portfolio website of Raj Kumar Khadka, a Computer Engineer from Nepal specializing in IT Infrastructure, AI/ML, and Platform Engineering.

🌐 **Live site:** [www.rajkumark.com.np](https://www.rajkumark.com.np)

## Tech Stack

- [Next.js 14](https://nextjs.org) — React framework
- [Tailwind CSS](https://tailwindcss.com) — Styling
- [Framer Motion](https://www.framer.com/motion) — Animations
- [Three.js / React Three Fiber](https://docs.pmnd.rs/react-three-fiber) — 3D elements
- [GSAP](https://gsap.com) — Advanced animations

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Deployment

Deployed on [Vercel](https://vercel.com) with automatic deploys on every push to `main`.
